# Web Design 1 – Assignment 1

## GitHub Setup and Introductory Web Page

## 📖 Project Overview

This project is part of the Web Design 1 (DCOMP204) course. The aim of this assignment is to introduce students to GitHub and basic web development using HTML.

The project involves creating a simple introductory web page and uploading it to a GitHub repository. This helps in understanding version control, file management, and basic webpage structure.



## 🎯 Objectives

* Create a GitHub account
* Join the GitHub Classroom repository
* Develop a basic HTML web page
* Upload and manage files using GitHub



## 💻 Project Structure

The project contains the following file:

* **index.html** – Main web page file



## 🌐 Web Page Content

The web page includes:

* Header section
* Full name and program
* Short biography
* Reason for studying web design
* Career goals
* Footer with personal details


## 🚀 How to View the Project

1. Download or clone the repository
2. Open the **index.html** file in any web browser


Screenshots of the web page and GitHub repository are included in the assignment documentation.



## 📚 Conclusion

This project demonstrates the basic use of HTML and GitHub for web development. It provides foundational knowledge that will be useful for more advanced web design tasks.
